# 社区行为准则（Code of Conduct）

本项目采用 Contributor Covenant 行为准则（简化版）。

## 我们的承诺

为建设开放、友好的社区环境，我们承诺在参与和贡献过程中尊重每一位参与者，不论其年龄、体型、可见或不可见的残障、民族、性别特征、性别认同与表达、经验水平、教育程度、社会经济地位、国籍、外貌、种族、宗教或性取向。

## 我们的标准

鼓励的行为包括但不限于：
- 使用友善、包容的语言
- 尊重不同观点与经验
- 以建设性的方式提出与接受反馈
- 聚焦对社区最有利的事情

不可接受的行为包括但不限于：
- 人身攻击、骚扰、歧视性言论
- 发布他人隐私信息（doxxing）
- 在任何场合进行不当的性暗示或冒犯行为

## 责任与执行

项目维护者有权并有责任在其认为不当的行为发生时采取适当且公平的纠正措施。

如需报告问题，请通过 `SECURITY.md` 中的联系方式联系维护者（或创建 issue 并标注为行为准则相关，避免暴露隐私信息）。
